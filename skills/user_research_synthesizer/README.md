# User Research Synthesizer
**ID:** `user_research_synthesizer`
**Category:** Research & Discovery

## Overview
Turns messy qualitative data (customer interviews, surveys, support tickets) into structured, quote-backed UX insights. It strictly prevents AI from inventing fictional user pain points by enforcing a "quote-first" extraction method.

## When to Trigger
- **Phase 1 (Problem Framing)**
- Trigger whenever you have raw user feedback and need a factual summary before defining the UX problem.
