# Creative Frontend Coder
**ID:** `creative_frontend_coder`
**Category:** Code Generation & Visual Execution

## Overview
The **Creative Frontend Coder** is your primary execution engine for translating design strategy and structural blueprints into actual, production-ready frontend code (React, HTML/CSS, Tailwind). 

Unlike the `UI Design Architect` (which focuses purely on structural logic and states), this skill focuses on the visual, aesthetic layer. 

## The Dual-Tier System
1. **Strict Compliance**: If fed an established Design System, it acts as a precise robot, adhering to hex codes and typography flawlessly.
2. **Creative Freedom ("Anti-Slop")**: If no Design System exists, it activates a strict creative manifesto forbidding generic, boring "AI" aesthetics (e.g., banning Inter/Roboto, purple gradients, and symmetrical grids) in favor of bold, memorable spatial composition, typographic choices, and high-impact motion.
