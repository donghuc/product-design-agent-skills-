# Reference & Benchmark Collector
**ID:** `reference_benchmark_collector`
**Category:** Research & Discovery

## Overview
Grounds AI-generated concepts in real-world precedent. This skill prevents hallucinations by forcing the AI to list VERIFIABLE competitor patterns before inventing new interaction models.

## When to Trigger
- **Phase 2 (Idea Exploration)**
- Run this immediately before the `concept_exploration_orchestrator` to provide grounded "reference material" for the design exploration.
