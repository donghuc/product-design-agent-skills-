# Product Design Workflow Orchestrator
**ID:** `product_design_workflow_orchestrator`
**Category:** Project Management & Meta-Routing

## Overview
This is the "Level 0" Meta-Agent. It acts as the project manager for the entire 5-Phase Product Design Cycle. Instead of doing the design work itself, it tracks your state and tells you exactly which of the 13 specialized skills you need to run next.

## When to Trigger
- **Start of a Project:** Ask it to help you kick off a new initiative from scratch.
- **Mid-Project Confusion:** Ask it "Where am I, and what skill should I use next?" and pass it your most recent `skill_handoff_schema.json`.
